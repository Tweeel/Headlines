# NewsApp
A simple News App using <a href="https://newsapi.org/">newsapi</a>
  get breaking news.
  see the article details using webview.
  save article that you like in the room db.
  search for news.
# Technologies
Kotlin<br>
MVVM architecture<br>
Room<br>
Corountine<br>
LiveData<br><br>
ViewBinding<br>
NewsApi -> https://newsapi.org/<br>
